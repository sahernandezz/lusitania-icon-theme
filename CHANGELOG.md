# Changelog

All notable changes to this project will be documented in this file.
This project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html)
and the format of [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] — 2026-05-20

### Added
- Initial release. 302 SVG icons inspired by IntelliJ *Material Icons*,
  recoloured with the [Lusitania](https://github.com/sahernandezz/lusitania-theme) palette.
- File icons for **176 extensions** (JS/TS, React, Vue, Svelte, Angular, Astro,
  HTML/CSS/SCSS, Python, Java/Kotlin/Scala/Groovy/Clojure, Go, Rust, Swift,
  Dart, Ruby, PHP, C/C++/C#, SQL, GraphQL, shells, Docker, fonts, images,
  audio/video, archives, documents…).
- Special icons for **129 file names** (`package.json`, `tsconfig.json`,
  `Dockerfile`, `docker-compose.yml`, `.env`, `.gitignore`, bundler / linter /
  test-runner configs, lockfiles, Cargo/Gradle/Maven/Composer…).
- Folder icons (closed + open) for **80 distinct names** with **179 aliases**.
- **56** language-ID mappings.
- Reproducible `build.py` (Python 3 stdlib only).
